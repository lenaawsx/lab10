Описание функционала

Приложение представляет собой галерею изображений, загружаемых через внешнее API.
Основной функционал включает:

Поле ввода для поиска изображений.

Грид-галерею с отображением картинок.

Ленту с ленивой подгрузкой (infinite scroll).

Просмотр изображения в формате Lightbox.

Сортировку и фильтры (по ориентации, порядку и т. д.).

Используемое API

Использовано Unsplash API для загрузки изображений по ключевым словам.

Примеры запросов
Поиск изображений:
https://api.unsplash.com/search/photos?query=cat&page=1&per_page=20&client_id=YOUR_API_KEY

Загрузка случайных изображений:
https://api.unsplash.com/photos/random?count=20&client_id=YOUR_API_KEY

Запрос с фильтрами:
https://api.unsplash.com/search/photos?query=city&orientation=landscape&order_by=latest&client_id=YOUR_API_KEY
